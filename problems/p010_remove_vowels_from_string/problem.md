# P010 Remove Vowels From String

## Summary

Given a string containing English letters, print the same string after removing every vowel.

One screenshot variant frames the same task as houses labeled with English letters, where only vowel-marked houses are renovated and the output should list the houses that will not be renovated.

## Matching Hints

- The statement lists the vowels as `a, e, i, o, u, A, E, I, O, U`.
- It asks to eliminate all vowels from a given string.
- The input is a single string.
- The output is the string after excluding all vowels.

## Notes

- Keep every non-vowel character in its original order.
- A set lookup is enough because only English letters are involved.
- The archived examples include `haveaniceday -> hvncdy` and `MynameisAnthony -> Mynmsnthny`.
